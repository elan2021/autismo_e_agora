# Technology Stack

## Core
- **Framework**: Next.js 16.2.4 (App Router)
- **Runtime**: Node.js
- **Language**: TypeScript 6.0.3
- **Frontend Library**: React 19.2.5

## Styling
- **Engine**: Tailwind CSS 4.2.4
- **Post-processor**: PostCSS 8.5.12
- **Utilities**: 
  - `clsx`: Utility for constructing className strings conditionally.
  - `tailwind-merge`: Utility for merging Tailwind CSS classes without conflicts.

## UI & Animation
- **Icons**: Lucide React 1.14.0
- **Animations**: Framer Motion 12.38.0

## Infrastructure
- **Deployment**: Optimized for Vercel (Next.js native)
- **Formatting/Linting**: ESLint with `eslint-config-next`
